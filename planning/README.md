## AI-DLC Process

Each Inception document follows the three-part structure:

1. **Press release** — what we're building, written as if already shipped
2. **Honest FAQ** — what could go wrong, what we're not building, hardest technical challenges
3. **Definition of Done** — numbered verifiable checklist; approval required before Construction code begins

The mob session approval record (date, attendees, hardest question + answer) is in `inception_v1.md`.
